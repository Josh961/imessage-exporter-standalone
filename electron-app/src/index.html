About iMessage Exporter
This software is open source and based on the code from ReagentX/imessage-exporter. This version is packaged as a standalone alternative, also open source, and can be found at Josh961/imessage-exporter-standalone. Both projects are licensed under the GPL-3.0 License. The installed executable is bundled with ImageMagick. ImageMagick is licensed under the ImageMagick License, which can be found at https://imagemagick.org/script/license.php.
